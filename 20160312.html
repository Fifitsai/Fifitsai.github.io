建置開發環境 (以windows為例)

█ 步驟 1：安裝Sublime Text3

檔案位置：https://www.sublimetext.com/3

下載Windows免安裝版portable version

解壓縮後開啟sublime_text

█ 步驟 2：安裝Node.js

檔案位置：https://nodejs.org/en/

下載安裝 Node 4.4.0

*測試是否安裝成功

搜尋CMD→enter→輸入node -v→enter，若出現版本，則表示是安裝成功

█ 步驟 3：安裝與設定常用Package

檔案位置：https://packagecontrol.io/installatio

複製下列文字

將剛剛複製的文字貼上去後按Enter，等待安裝

到Sublime Text3按Ctrl + ~ 會出現白色輸入框，如下圖

安裝完若出現下圖錯誤訊息，按確定略過即可

*測試是否安裝成功

在Sublime Text3按(Ctrl+Shift)+P，若出現下列畫面即表示成功

接著開始安裝實用的Package

*htmlcssjs Prettify_自動縮排

(Ctrl+Shift)+P → 輸入pci → enter → 輸入htmlcssjs Prettify →點擊安裝

*emmet輸入短碼，再按tab鍵 自動生成html碼

(Ctrl+Shift)+P → 輸入pci → enter → emmet →點擊

安裝完成後會出現下列檔案

接著選單點選Preferences→Package Settings→HTML/CSS/JSPrettify→set`node`Path

出現下列檔案畫面後

將“format_on_save”: false“改為“format_on_save”: true“

若剛剛搜尋CMD→node -v 確認版本，有成功

則可將node_path”: “C:/Program Files/nodejs/node.exe” 改為 node_path”: “node”

█ 步驟 4：測試開發環境是否建置成功

在Sublime Text3開新檔案

另存新檔為test.html

測試htmlcssjs Prettify

輸入" < h 選擇html → enter

出現下列html程式碼，存檔(Ctrl+S)後html程式碼會自動排列

測試emmet

1.輸入html → 按Tab，會自動生成html程式碼

大功告成!!
